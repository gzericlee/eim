### 单机部署

暂无参数说明，默认配置可直接部署。

1. 依赖服务
```shell
 ./deploy/docker-compose/up.sh all
```
2. 编译EIM服务
```shell
 ./build/build.sh
```
3. 启动服务
```shell
 cd ./build/dist 
```
